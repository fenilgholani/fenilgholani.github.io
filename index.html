Improving Computer Performance by Pipelining

This project focuses on improving the performance of the computer with the help pipelining, forwarding and overcoming some of the Data Hazards, Structural Hazards and Conditional Hazards. This idea is implemented in C programming language. This program is also used to determine dependence and accordingly add stalls to the cycle to prevent Structural Hazards. Data Hazards are reduced with the help of Forwarding of the pipeline. Conditional Hazards are reduced by predicting the jump statement prior the Decode Phase.
